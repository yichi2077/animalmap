# WildAustralia — 产品需求文档 PRD v2.0
> 状态：定稿草案 · 版本：v2.0 · 日期：2026-03
> 变更摘要：重新定义核心地图交互模型、升级动物可视化方案（三层系统替代Emoji）、确认完整28种动物数据集、新增地图风格规范

---

## 一、产品定位（不变）

**一张会呼吸的地图**——用时间轴带用户穿越500年澳大利亚野生动物的迁徙、消亡与挣扎。

核心差异化：**时间旅行 × 情感叙事 × 空间演化**，而非静态动物百科。

---

## 二、核心设计原则（v2.0 新增）

### 原则 1：地图即完整宇宙
地图占据整个浏览器视口。没有割裂的页面结构，没有独立的「详情页」。所有信息在同一张地图上发生。侧边详情面板浮于地图之上，不替代地图。

### 原则 2：动物始终可见
用户不需要进入任何子页面才能看到动物。打开网站即可在全国地图上看到所有物种的分布。「进入州区」的操作是过滤与聚焦，而非导航跳转。

### 原则 3：时间轴是主叙事引擎
时间轴不是筛选器，是故事的进度条。拖动时间轴时，整张地图在演化——栖息地色块在扩张或收缩，动物剪影在生长或凋零，历史事件在爆发或平息。

### 原则 4：用空间感替代符号感
放弃 Emoji 点状符号。改用三层可视化系统，让物种演化有面积感、有重量感、有消逝感。

### 3.2 地图缩放与聚焦

#### 全国视图（默认状态）
- 所有州边界可见
- 28 种动物全部以三层系统显示
- 支持鼠标滚轮 / 双指捏合缩放
- 支持拖动平移（不触发聚焦）

#### 州区聚焦（直接点击州区触发）
**无需文字按钮，直接点击地图上的州区域即可触发**

**动画时序**（以点击 QLD 为例）：
```
t=0ms    用户点击 QLD 区域内任意位置
t=0ms    其余州区 opacity: 1→0.2（400ms ease）
t=0ms    Mapbox flyTo() 启动：地图飞向 QLD 中心，zoom+2（600ms）
t=400ms  QLD 内动物图层亮度 +30%，清晰度提升
t=500ms  QLD 生态区细分多边形淡入
t=600ms  右侧物种列表面板从右边缘滑入（该州特有物种）
t=600ms  动画完成
```

**退出聚焦方式**：
- 点击地图空白海洋区域
- 点击右上角「× 全国视图」按钮
- 向外缩放到全国级别 zoom
- 触发 flyBack 动画 + 其余州区恢复亮度

#### 物种高亮模式
触发：点击任意动物剪影 / 搜索框命中物种
- 该物种：栖息地多边形亮度 ×2，剪影放大至 1.4×
- 其余所有物种：透明度降至 15%
- 顶部浮出物种快速信息条

## 四、三层动物可视化系统（核心创新）

### 为何放弃 Emoji

Emoji 是离散的点状符号，存在三个根本缺陷：
1. **无范围感**：无法表达物种的「栖息地领土」，无法感受到「这片土地正在失去什么」
2. **无密度感**：无法编码种群数量信息，考拉9万只和袋狼最后1只在地图上看起来一样
3. **无消逝感**：消失只是 `opacity: 0`，缺乏情感冲击

### 层级架构

```
Layer 3（顶层）：消亡粒子系统         ← Canvas 2D，独立图层
Layer 2（中层）：SVG 动物剪影标记     ← Mapbox Symbol Layer
Layer 1（底层）：栖息地范围多边形     ← Mapbox Fill Layer
地图底层：GeoJSON州界 + 羊皮纸底图   ← Mapbox GL JS
```

### Layer 1：栖息地范围多边形

每种动物对应一个（或多个）半透明色块多边形，覆盖在地图上，代表其栖息地分布范围。

**颜色规则**（按濒危等级）：
- LC 无危：`rgba(90, 120, 60, 0.10)` 橄榄棕绿
- NT 近危：`rgba(160, 130, 50, 0.12)` 暖琥珀
- VU 易危：`rgba(190, 110, 40, 0.13)` 橘棕
- EN 濒危：`rgba(200, 70, 40, 0.14)` 深橙红
- CR 极危：`rgba(200, 40, 30, 0.16)` 危机红
- EX 已灭绝（存续年份）：`rgba(80, 60, 40, 0.12)` 深棕

**时间轴联动动画**：
- 每经过关键历史节点 → 对应物种多边形按 clip-path 动画向核心区域收缩
- 灭绝年份到达 → 多边形在 1.5s 内消失（配合 Layer 3 粒子）
- 未来预测区 → 透明度降至 0.4，边框变虚线，表达不确定性

**多边形叠加效果**：
多物种分布重叠区域颜色自然叠加，形成生物多样性热点视觉——热点区（如 QLD 东北热带雨林、TAS）颜色更深，直觉上感受到「这里动物最多」。

### Layer 2：SVG 动物剪影标记

替代 Emoji，使用精细的 SVG 动物剪影（来源：phylopic.org CC0 授权）放置在每种动物的核心栖息地中心点。

**尺寸编码种群规模**：
```
种群规模 → 剪影大小
>100万   → 36px
10-100万 → 28px
1-10万   → 22px
1-10千   → 16px
<1千     → 12px（濒危边缘，视觉上接近消失）
```

**颜色系统**（基于羊皮纸底图配色）：
- 剪影填充色：`#f0e4cc`（羊皮纸白）
- 轮廓线颜色随状态：
  - LC：`#5a4030`（深棕）
  - NT：`#c89040`（暖金）
  - EN：`#d05030`（橙红）+ 低频脉冲光环动效（2.5s周期）
  - EX（存续期）：`#706860`（冷灰）

**交互行为**：
- 悬停 → 剪影 scale 1.3×，底部出现半透明名称气泡（中文名 + 状态徽章）
- 点击 → 物种详情面板滑出，进入物种高亮模式

**时间轴联动**：
- 种群增长/下降 → 剪影大小平滑过渡（CSS transition 800ms）
- 灭绝年份 → 触发 Layer 3 粒子爆散 → 剪影在爆散后 0.5s 消失
- 殖民前（<1788）→ 剪影带有棕褐色调 filter，表达「推断性数据」

### Layer 3：消亡粒子系统

**触发时机与视觉设计**：

**物种灭绝粒子（点状冲击）**：
- 触发：时间轴滑过物种灭绝年份
- 效果：剪影原点爆出 60-80 个粒子，向外圆形扩散，速度由快到慢，透明度渐至0
- 持续：2.5 秒
- 颜色：与物种相关（袋狼：灰白 `#c8c0b0`，海龟：蓝绿 `#40a8c0`）
- 爆散结束后，地图上仅留下灰色「消亡遗迹」轮廓圈（半透明，直径 40px，表示「曾经在这里」）

**山火全图粒子风暴**（特殊效果，仅 2019 年触发）：
- 触发：时间轴拖到 2019 年任意一天
- 效果：全图从 QLD/NSW 边境向西南方向扩散橙红色粒子流，约 400 个粒子
- 持续：3 秒，后慢慢消散
- 配合：全图轻微橙色遮罩叠加（opacity 0.08，持续 5s 再消散）
- 同期触发：考拉、袋獾、绿海龟等多个物种的剪影缩小动画

**持续渗出粒子（濒危物种专属）**：
- 濒危（EN/CR）物种在 NOW 节点上，剪影会持续渗出 3-5 个粒子（周期 4s）
- 粒子向上漂浮并消散，类似「烟雾」效果
- 视觉隐喻：「正在缓慢消逝中」

## 五、地图风格规范（v2.0）

### 整体风格：高保真复古航海地图

参照：18世纪英国皇家地理学会测绘风格（库克船长时代的澳大利亚测绘图）
目标感受：**有历史重量、有探险感、有凝重叙事质感**

### 色彩系统（全部硬编码，不随深色模式变化）

```css
/* 底图基础色 */
--ocean:       #1a3a5c    /* 深靛蓝海洋 */
--land:        #d4b896    /* 羊皮纸陆地 */
--land-dark:   #c4a882    /* 山地/高原略深 */
--coast:       #7a5830    /* 海岸线深棕 */
--state-line:  #8a6840    /* 州边界棕色 */
--river:       #4a7a9a    /* 内陆河流钢蓝 */
--desert:      #c89060    /* 内陆沙漠暖棕 */
--reef:        rgba(26,158,180,0.2)  /* 大堡礁淡青 */

/* 地图装饰 */
--grid-line:   rgba(255,255,255,0.05)  /* 经纬度网格 */
--state-label: rgba(100,70,40,0.45)    /* 州名刻字 */
```

### 地图纹理细节

- **纸张颗粒感**：SVG `feTurbulence` 噪声滤镜叠加，opacity 8%
- **海岸深度线**：沿澳大利亚海岸线向外每 2px 绘制一条平行等值线（共 5 条），颜色 `rgba(30,80,140,0.15)`，模拟18世纪海图测深线
- **地形晕渲**：大分水岭区域（东部山脉）用手绘风格 hachure 纹理表示，无真实高程数据
- **装饰元素**：
  - 右下角罗经花（compass rose），opacity 12%
  - 地图边框：双线矩形边框，内线细，外线稍粗，羊皮纸风格
  - 图例区域：左下角放置比例尺 + 图例，手绘风格排版

### 字体系统

| 用途 | 字体 | 颜色 |
|------|------|------|
| 品牌标题 | Playfair Display 700 | `#c8902a` 金色 |
| 地名标签 | IM Fell English（Google Fonts） | `#7a5830` |
| 时间轴年份 | JetBrains Mono 500 | `#e8d8b0` |
| 动物气泡标签 | Lora 400 | `#d4c4a0` |
| UI 控件文字 | Lora 400 | `#c4b090` |
| 动物学名 | Source Serif 4 400 italic | `#9a8060` |
| 濒危徽章 | JetBrains Mono 500 | 按状态色 |

## 七、时间轴规范（v2.0）

### Era 分段与配色

| Era | 年份范围 | 时间轴颜色 | 地图状态 |
|-----|---------|-----------|---------|
| 殖民前 | 1500–1787 | 苔绿 `#5a8a5a` | 动物最丰富，剪影最大 |
| 殖民初期 | 1788–1849 | 暖棕 `#a07840` | 开始有物种退出 |
| 扩张时代 | 1850–1899 | 赭石 `#b06030` | 色块可见收缩 |
| 联邦工业化 | 1900–1949 | 深橙 `#c05028` | 多物种加速退缩 |
| 现代 | 1950–1999 | 沙金 `#b89040` | 保护运动兴起 |
| 当代危机 | 2000–2024 | 警示红 `#d04020` | 气候变化叠加 |
| NOW | 2024 | 白 `#ffffff` | 全亮特殊状态 |
| 未来预测 | 2025–2080 | 雾蓝 `#4a6a9a` | 透明+虚线 |

### 时间轴 UI 元素

```
[⏮]  ━━━━━━━━━━━━━━━━━━━━━━━━●━━━━━━━━━···················  [⏭]
    1500  1788  1850  1900  1936  2000  2019 NOW  2050  2080
                            ★    ★                ★
    ★ = 重大历史事件节点，悬停弹出事件卡片

底部统计栏：
    当前年份：1936    现存物种：24    濒危物种：8    已灭绝：2
```

- `⏮` 快速跳回 1500（推断自然时代起点）
- `⏭` 快速跳到 NOW
- 底部统计数字随时间轴实时动态更新（count-up/count-down 动画）
- 每年为最小单位（拖动精度 1 年）

## 八、动物详情面板（v2.0）

### 结构与内容

```
┌─────────────────────────────────────────────────────┐
│ ← 返回                                    ✕ 关闭   │
```

## 九、技术选型（v2.0 确认）

| 层级 | 技术 | 理由 |
|------|------|------|
| 框架 | React + TypeScript | 组件复用，类型安全 |
| 地图引擎 | **Mapbox GL JS** | 支持自定义风格、GPU渲染、flyTo动画、GeoJSON图层 |
| 地图风格 | Mapbox Studio 自定义样式 | 实现羊皮纸风格 |
| 地图边界数据 | GADM GeoJSON（免费） | 精确州级边界 |
| 动物分布多边形 | 自定义 GeoJSON | MVP 手动描绘，后期接 GBIF |
| 时间轴 | D3.js 自定义 | 精确年份→地图状态联动 |
| 粒子系统 | Canvas 2D API（原生） | 轻量，不影响地图层性能 |
| 动物剪影素材 | phylopic.org（CC0）+ 自制 | 高质量科学剪影 |
| 剪影渲染 | Mapbox Symbol Layer（SVG） | GPU加速，随缩放级别自动调整 |
| 地图缩放动画 | Mapbox flyTo() | 原生支持平滑飞行动画 |
| 面板动画 | Framer Motion | 滑入/滑出手感 |
| 样式 | Tailwind CSS | 工具类快速开发 |
| 音频 | xeno-canto API + Web Audio | 真实叫声 + 波形可视化 |
| 内容 | Claude API 预生成静态 JSON | 构建时生成，零运行时延迟 |
| 部署 | Vercel | CDN + Serverless |

### Mapbox 免费额度
- Web Maps：每月 50,000 map loads（够 MVP 阶段）
- 若超出：$0.50 / 1000 loads
- 地图风格创建：完全免费

## 十一、性能要求

| 指标 | 目标值 | 策略 |
|------|-------|------|
| 首屏加载 | < 3s | 地图瓦片按需加载，静态JSON预渲染 |
| 时间轴拖动响应 | < 16ms | Mapbox GPU渲染，Canvas粒子独立线程 |
| 地图缩放动画 | 60fps | Mapbox flyTo原生支持 |
| 面板滑入 | < 300ms | Framer Motion will-change |
| 粒子峰值 | < 100个/帧 | 灭绝事件上限80粒子，山火最多400粒子（分批） |
