## Design Context

### Users
自然爱好者和普通大众——对澳洲野生动物感兴趣的中文用户，不需要专业背景。他们希望通过一张"会呼吸的地图"了解澳大利亚野生动物的时空变迁故事，体验翻开自然绘本的感觉。全年龄段友好，从好奇的孩子到关心自然的成年人都能轻松使用。

### Brand Personality
**三个关键词：温暖、治愈、故事感**

- 语调：像一位温柔的自然讲述者，用平静的声音讲述生命的故事
- 情感目标：用户应感到自己在观看一张呼吸着的地图，而非使用一个工具；像翻开一本自然绘本，观察一个大陆上的生命故事
- 不是冰冷的数据分析，不是严肃的学术报告，而是有温度的自然叙事

### Aesthetic Direction

**视觉基调：** 治愈系手绘绘本 + 羊皮纸质感 + 低饱和大地色系

**核心视觉元素：**
- 羊皮纸纹理背景（#fdf8f0 基底，微妙径向渐变模拟纸张质感）
- 手绘线条感（SVG 位移滤镜模拟不规则笔触）
- 水彩风格色彩（低饱和、温暖、自然）
- 大圆角面板（1rem-2rem）
- 暖色调阴影（earth 色系 rgba）
- 呼吸感动画（缓慢、轻柔、有生命力）

**色彩系统：**
- 基底色：Parchment 系列（#fefdfb → #f5ead5）
- 主色调：Earth 系列（#7a6b52 文字，#b8a88a 辅助）
- 自然色：Leaf（#7da56c）、Ocean（#9ec3d8）、Coral（#e08a58）
- 辅助色：Mint（#64ba9c）、Sky（#6cb3d9）、Sand（#e2cda6）

**字体系统：**
- 展示字体：Noto Serif SC（标题、强调）
- 正文字体：Noto Sans SC（界面、正文）
- 手写字体：Ma Shan Zheng（装饰性元素）

**参考方向：** 自然绘本、手绘地图集、博物馆互动展览

**反面参考（绝对避免）：**
- 深海蓝/纯黑科技感
- 赛博朋克风格
- BI 数据仪表盘
- 典型 SaaS 后台风格
- 强分析感 GIS 可视化
- 超写实自然纪录片风格
- 高饱和玩具感儿童设计
- 任何 emoji 使用

### Design Principles

1. **地图即页面** — 地图不是被压缩在小区域里的组件，它就是页面本身。所有 UI 元素都是地图上的叠加层。
2. **故事优先，分析其次** — 每个设计决策都应服务于情感共鸣和叙事体验，而非数据展示效率。用户在"阅读"故事，不是在"分析"数据。
3. **克制的动画** — 动画只在服务体验时出现：呼吸感（生命力）、渐入渐出（自然过渡）、弹性（温暖反馈）。绝不为炫技而动。
4. **温暖而非冰冷** — 每个交互都应传递温度：圆角而非直角，暖影而非冷影，大地色而非科技蓝，手绘感而非精确几何。
5. **全年龄可达** — 界面足够简洁让孩子能探索，内容足够丰富让成人有收获。基础无障碍：颜色对比度达标、键盘可导航、状态不仅靠颜色区分。

### Animation Philosophy
- 呼吸动画（gentle-breathe）：4s 周期，模拟生命体的自然节律
- 漂浮动画（gentle-float）：5s 周期，轻柔的上下浮动
- 幽灵动画（fade-ghost）：6s 周期，用于濒危/低种群物种的若隐若现
- 面板进入：Framer Motion spring（stiffness: 280, damping: 26）
- 灭绝物种：grayscale(100%) + opacity(0.4)，线稿/幽灵状态
- 所有动画应尊重 prefers-reduced-motion 偏好

### Component Patterns
- 面板风格：大圆角 + 羊皮纸渐变背景 + 暖色阴影 + 微妙边框
- 不使用毛玻璃效果（glass-morphism）
- 交互反馈：scale + opacity 的弹性过渡
- 状态变化：平滑的 CSS/Framer Motion 过渡，避免突兀跳变
